# SiH-FabSpector

This project is a web application designed for fabric defect detection and analysis. It provides a comprehensive dashboard to visualize and manage the quality control process in fabric manufacturing.

## 🚀 Features

- **Dashboard:** An overview of key metrics, including total rolls inspected, defect rates, and recent activity.
- **Live Inspection:** A real-time view of the fabric inspection process.
- **Defect Gallery:** A gallery of detected defects for review and analysis.
- **Roll Reports:** Detailed reports for each roll of fabric.
- **Color Clustering:** Analysis of color variations in the fabric.
- **Settings:** Configuration options for the application.
